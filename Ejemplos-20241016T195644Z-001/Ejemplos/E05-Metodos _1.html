Vue - Directiva v-html

v-html permite insertar contenido HTML dinámico en un elemento del DOM.

Es importante tener en cuenta que cuando utilizas v-html, debes tener cuidado con la seguridad, especialmente si el contenido HTML proviene de fuentes externas o usuarios no confiables, ya que puede presentar riesgos de seguridad como ataques XSS (Cross-Site Scripting). Es recomendable validar y escapar adecuadamente el contenido HTML para mitigar estos riesgos.
