Vue - Acceder a elementos mediante $ref

Creamos una app con Vue, y utilizamos un contenedor div para renderizar dinámicamente los datos proporcionados por esta app.

Si necesitamos acceder a elementos del DOM, las instancias de Vue.JS cuentan con diversas propiedades, una de ellas es $refs.

ref es un objeto, dentro de él se almacenan todos los elementos del DOM que cuentan con el atributo ref. El atributo ref es similar a proporcionar un ID al elemento.
